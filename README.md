Hello, Delivery! (The Maasdam Project)

Скрипт создания планирования окон доставки для интернет-магазинов. Называется в честь дырок от сыра маасдам.

/www/contract.html - Скрипт бэк-офиса для управления зонами доставки и расписанием работы службы.

Описание табов:
- Разметка зон - позволяет добавлять, удалять, редактировать и размечать зоны доставки, называть их и назначать стоимость доставки по району
- Шаблон расписания - позволяет задать базовый недельный шаблон расписания для каждой зоны доставки, задать максимальную ёмкость (количество доступных доставок) для временных окон "утро", "день" и "вечер"
- Редактор расписания - позволяет управлять календарём расписания доставок (календарь создаётся на основе недельного шаблона)


/www/maasdam.html - Скрипт клиентской части магазина. По клику на карту или при определении адреса из поиска выводит бабл в котором клиент может выбрать ближайшее удобное для него окно доставки.

/api.txt - описание серверного API для управления и запроса зон доставок

/structure.txt - описание структуры БД для хранения данных


Серверную часть не выкладываю, так-как была сильно завязана на закрытый фреймворк.
